## 控制台输出对象的响应式

```
 const obj = { a: 1 };
      console.log("obj", obj); //  obj {a: 1}
      console.log("json", JSON.stringify(obj)); //json {"a":1}

      setTimeout(() => {
        obj.a = 2;
        console.log("timerObj", obj); // timerObj {a: 2}
        console.log("timerJson", JSON.stringify(obj)); // timerJson {"a":2}
      }, 0);
```

调试时候多用json处理避免响应式
